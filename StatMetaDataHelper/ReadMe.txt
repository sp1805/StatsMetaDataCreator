****************Steps before Running this code********************
1. Make a folder "StatXMLHelper" in C:\StatXmlHelper
2. Copy all your dropdowns xml file to this location
3. Remove the state specific naming convention from the file, eg, States.OR_OSWEGO.xml to States.xml
4. While running code manually enter the name of file
5. You will get the desired xlxs file in the same folder as "new.xlsx"
